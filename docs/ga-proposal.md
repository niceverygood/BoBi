## 2. 플랫폼 한 줄 소개

> **GA 소속 설계사 1인의 영업·고객관리·청약 사전점검을 5분 안에 끝내는 AI 컨설팅 도구.**
>
> 본인 동의 기반 진료 데이터 분석 + 자동 알림톡 CRM + 보장 갭 시각화를 한 앱에서.

---

## 3. 시장 또는 사용자 문제

### GA 본사가 직면한 4가지 문제

| 문제 | 영향 |
|---|---|
| **소속 설계사 영업 효율화 도구 부재** | 1,000~5,000명 설계사 평균 분석 시간 3시간/건 → 영업 시간 잠식 |
| **청약서 고지누락 → 보험금 분쟁** | 분쟁 1건당 평균 합의금 200~500만원 + GA 이미지 타격 |
| **신인 설계사 교육 비용** | 1명당 평균 6개월 교육 + 이탈률 40% (도구 없으면 더 심함) |
| **자회사 GA 대비 차별화 부족** | 한화·삼성생명금융서비스 등 본사 디지털 도구 보유 GA와 격차 ↑ |

### 설계사 1인이 매일 겪는 4가지 페인포인트

1. **고객 분석에 시간이 너무 오래 걸린다** — 1명당 평균 3시간 (수기 정리)
2. **클로징 무기가 부족하다** — "왜 보험이 필요한가" 말로만 설득
3. **영업 기회를 놓친다** — 갱신·면책·생일 등 컨택 타이밍 수기 관리
4. **데이터·시스템 파편화** — HIRA 진료내역 / 건강검진 / 보험 가입 정보 따로따로

→ **GA 본사·설계사 양쪽 페인포인트를 한 도구로 해결**.

## 4. 플랫폼의 해결 방향

### 4가지 핵심 원리

1. **본인 동의 진료 데이터 + AI** — 5년치 진료내역을 5분에 분석 (CODEF API)
2. **자동 CRM 알림톡** — 갱신·면책·생일 자동 발송, 영업 기회 0% 미스
3. **보장 갭 시각화** — "만약 그 질환 생기면" 시뮬레이션 한 장
4. **카카오 알림톡 = 설계사 본인 이름** — 자연스러운 브랜딩

### GA 본사 관점의 가치

- **설계사 생산성 ↑** (3시간 → 5분 = 영업 시간 36배 확보)
- **분쟁 발생률 ↓** (청약 전 고지누락 사전 점검)
- **교육 비용 ↓** (도구 자체가 표준 분석 흐름 제공)
- **차별화 ↑** (자회사 GA 수준 디지털 무장)

## 6. GA 설계사 구조 (보비가 fit하는 위치)

```
[GA 본사]
   │
   ├─ 영업총괄·전략기획
   │     └─ 보비 단체 라이선스 계약 + 사용량 대시보드
   │
   ├─ 지점·지사 (수십~수백 개)
   │     └─ 지점장 — 소속 설계사 모니터링
   │
   └─ 소속 설계사 (1,000~5,000명)
         │
         └─ 보비 앱 사용 (분석·CRM·알림톡)
                │
                └─ 고객 (분석 결과 카카오 알림톡 수신)
```

**보비 통합 포인트**:
- **본사**: 단체 라이선스 + 통계 대시보드 (사용량·분쟁 감소율)
- **지점**: 지점별 사용 KPI (선택)
- **설계사**: 일상 도구로 사용
- **고객**: 알림톡 수신·리포트 확인

## 7. 플랫폼 작동 방식

### 설계사 일상 흐름 (1명 분석 시나리오)

```
1. 고객 정보 입력 (이름·연락처·생년월일)
       ↓
2. 본인인증 동의 링크 발송 (카카오 알림톡)
       ↓
3. 고객 동의 → CODEF API로 5년치 진료내역 자동 조회
       ↓
4. AI 분석 (3~5분): 주요 진단·복용 약물·고지 누락 점검
       ↓
5. 리포트 자동 생성: 진료정보 / 위험도 / 가상영수증 / 미래의 나
       ↓
6. 카카오 알림톡으로 고객에게 발송 (발신자 = 설계사 이름)
       ↓
7. 갱신·면책·생일 자동 알림 시작 (매일 새벽 cron)
```

### GA 본사 관리자 흐름

- 본사 어드민 페이지 → 소속 설계사 사용 현황 / 분석 건수 / 알림톡 발송 통계
- 분쟁 발생 시: 청약 전 고지누락 점검 로그 확인 → GA 책임 면책 근거

## 8. 핵심 기능 (5대 + α)

### 핵심 5대 기능

| 기능 | 설명 | GA 가치 |
|---|---|---|
| **진료정보 분석** | 5년치 진료내역 자동 정리 + 고지 누락 점검 | 청약 분쟁 사전 차단 |
| **위험도 리포트** | 일반인 대비 개인 질환 위험 배율 | 추가 가입 클로징 무기 |
| **가상 사고영수증** | "만약 그 질환 생기면" 보장 갭 시각화 | 설득력 있는 보험금 갭 제시 |
| **CRM 자동 알림톡** | 갱신·면책·감액·생일 자동 발송 | 영업 기회 0% 미스, 고객 LTV ↑ |
| **미래의 나** | 연령대별 의료비 변화 시뮬레이션 | 젊은 고객 클로징 |

### 추가 (α)

- **카카오 알림톡 통합 발송**: 4종 리포트 모두 알림톡 직접 전송 가능
- **설계사 본인 이름 브랜딩**: 발신자 = 설계사 → 고객 신뢰 ↑
- **고객 카드 CRM**: 보험 가입 정보·메모·이력 통합 관리
- **PDF 리포트 다운로드**: 인쇄·메일 첨부 가능

## 10. 기존 방식 대비 차별점

| 항목 | 기존 방식 | 보비 |
|---|---|---|
| 진료내역 분석 | Excel·종이 수기 정리 (3시간) | AI 자동 (5분) |
| 위험 설명 | 설계사 말로만 | 데이터 기반 N배 시각화 |
| 보장 갭 제시 | 막연한 설명 | 숫자(예: "2,640만원 부족") |
| 갱신·면책 관리 | 수기 캘린더 | 매일 새벽 자동 알림톡 |
| 고객 컨택 | 개별 메신저·전화 | 카카오 알림톡 통합 (설계사 브랜딩) |
| 청약 고지누락 점검 | 사후 발견 → 분쟁 | 사전 자동 점검 → 분쟁 차단 |
| 신인 교육 | 1:1 OJT 6개월 | 도구 자체가 표준 흐름 제공 |

## 12. 수익 모델 / 비즈니스 모델

### 3-Layer Revenue

| Layer | 대상 | 가격 |
|---|---|---|
| **Layer 1 — 개인 SaaS** | 설계사 1인 | 베이직 19,900원/월 · 프로 39,900원/월 |
| **Layer 2 — GA 단체 라이선스** | GA 본사 | **GA 규모별 협상** (인당 8,900~13,000원) |
| **Layer 3 — API 라이선싱** | 핀테크·자회사 GA | MAU 기반 (협의) |

### GA 단체 라이선스 단가 (가이드)

> 💡 **무제한 사용** 기준 단가 (프로 플랜 39,900원의 **50% 할인**).
> 베이직(분석 30회 한도)이 아닌 프로(무제한) 기능을 단체 가격으로 제공.

| GA 규모 | 인당 단가 | 월 비용 | 연간 비용 | vs 개인 프로 (39,900원) |
|---|---|---|---|---|
| 50명 | **19,950원** | 997,500원 | 11,970,000원 | **50% 할인** |
| 100명 | **17,950원** | 1,795,000원 | 21,540,000원 | 55% 할인 |
| 200명 | **15,960원** | 3,192,000원 | 38,304,000원 | 60% 할인 |
| 500명+ | **별도 협상** | 협의 | 협의 | 65%+ 할인 |

### 무제한 라이선스에 포함되는 것
- ✅ 진료정보 분석 **무제한** (개인 베이직은 30건/월 한도)
- ✅ 위험도 리포트 · 가상 사고영수증 · 미래의 나 모두 무제한
- ✅ CRM 자동 알림톡 (갱신·면책·감액·생일)
- ✅ 본사 어드민 대시보드 (사용량·KPI 모니터링)
- ✅ 우선 고객 지원 (영업일 4시간 이내 응답)

### 장기 협력 옵션

- **연간 계약** 시 추가 10~15% 할인
- **3년 계약** 시 추가 기능 우선 제공 + 본사 브랜드 커스터마이징 + 화이트라벨
- **API 임베드** (Layer 3) 시 본사 자체 앱에 보비 분석 엔진 통합 (별도 협상)

## 13. 운영 및 품질 관리 체계

### 데이터·법무 안정성
- **CODEF API 공식 연동** — 본인 동의 기반 합법적 데이터 조회 (마이데이터 사업자 연계는 향후 확장)
- **약관·법무 사전 검토** (`docs/insurance-db-legal-questions.md`) — 보험업법·개인정보보호법 사전 확인
- **개인정보처리방침 + 환불 정책** 공시 — 전자상거래법 부합

### 시스템 운영
- **Vercel + Supabase** (글로벌 표준 인프라, 99.9% SLA)
- **Sentry** 오류 모니터링 + **PostHog** 사용 분석
- **자체 디자인시스템 §11.3** — 의료/금융 단정 표현 금지·정직 톤

### 고객 지원
- **평일 10:00~18:00** 카카오톡·이메일 지원 (support@bobi.co.kr)
- **1:1 문의** 평균 응답 24시간 이내
- **장애 발생 시** 일할 환불 + 즉시 알림

### 품질 KPI (분기별 모니터링)
- 분석 정확도 (사용자 만족도 NPS)
- 알림톡 발송 성공률 (목표 99%+)
- 시스템 가동률 (목표 99.9%)
- 환불률 (목표 5% 미만)

## 14. 성장 전략 및 확장 가능성

### 제품 확장 로드맵 (가장 중요한 변화)

```
2026.현재 → 2026.Q4         → 2027~                    → 2028~
─────────  ─────────────────  ─────────────────────────  ─────────────
[Phase 1]  [Phase 2]          [Phase 3]                  [Phase 4]
진료·건검   마이데이터 사업자    내 보험 다 보여            AI 보험상품

─────────  ─────────────────  ─────────────────────────  ─────────────
CODEF       마이데이터 정식      모든 가입 보험            보비 추천
본인 동의   사업자 라이선스      한 화면 통합              + 청약 연결
```

- GA 설계사가 고객의 **모든 가입 보험을 한 화면**에서 확인

**Phase 3 — "내 보험 다 보여" 출시 (2027.H1)**
- 마이데이터 연계 후 즉시 가능한 핵심 기능
- 고객이 어떤 보험을 어디에 얼마나 들고 있는지 한눈에
- 중복 보장·과다 보장·보장 갭 자동 분석

**Phase 4 — AI 보험상품 추천 (2027.H2~2028)**
- 진료 데이터 + 보유 보험 + 위험도 → AI가 최적 상품 추천
- GA 본사와 제휴해 추천 → 청약 연결 (수수료 모델)
- B2B SaaS + 추천 수수료 = **2-Layer 매출 구조**

### 단기 (2026)
- **소속 설계사 1만명 달성** (이종인 영업이사 1,200+ 채널 + GA 무상배포)
- **알림톡 발송 월 100만건** (CRM 자동화 본격 가동)
- **유료 전환율 12% → 18%** (제품 완성도 강화)
- **마이데이터 사업자 연계 협상** 시작

### 중기 (2027~2028)
- **"내 보험 다 보여"** 정식 출시 (마이데이터 기반)
- **AI 보험상품 추천** 베타 → 정식 (청약 연결 수수료 모델)
- **API 라이선싱 (Layer 3)**: 핀테크 GA·자회사 GA 임베드
- **일본 진출**: 일본 보험설계사 100만, 디지털 도구 부재 시장 선점
- **B2B 화이트라벨**: GA 자체 브랜드로 보비 엔진 제공

### 장기 (2029~2031)
- **동남아 진출** (베트남·태국)
- **시리즈 A·B 확보 후 IPO** (코스닥, 비교: 인카금융서비스 시총 1,500억)
- **보험사 자회사 GA M&A 후보** (한화·삼성생명 자회사 GA가 인수 가능성)

### GA 본사가 얻는 미래 가치
- ✅ **2026.Q4**: 마이데이터 연계로 고객 보유 보험 통합 → **중복 보장 사전 발견** + 추가 가입 기회
- ✅ **2027**: 보비가 자동 추천한 상품을 GA 채널로 청약 연결 → **신규 매출 채널** + 수수료
- ✅ **장기**: 보비 데이터 + GA 채널 = 정밀 타게팅 영업 (지금까지 없던 마케팅 도구)

## 15. 도입 / 입점 / 이용 절차

### GA 본사 도입 5단계 (4~6주)

| 주차 | 단계 | 주요 활동 |
|---|---|---|
| **1주차** | 본사 협의 | 단가 협상 + 계약서 초안 검토 + 데이터 처리 위탁 계약 |
| **2주차** | 계약·라이선스 발급 | 단체 라이선스 활성화 + 본사 어드민 계정 발급 |
| **3주차** | 교육·자료 전달 | 설계사용 사용 가이드 + 본사 스태프 사전 교육 (1회 ZOOM) |
| **4주차** | 점진적 가입 | 지점장 → 우수 설계사 → 전체 단계적 온보딩 |
| **5~6주차** | 정상 운영 | 사용량 모니터링 + 본사 KPI 보고 + 피드백 반영 |

### 무상 파일럿 옵션
- **30명 이하 GA**: 1개월 무상 사용 → 만족 시 정식 계약
- **50명 이상 GA**: 1주일 본사 임원 시연 + 2주 우수 설계사 베타 → 정식 계약
