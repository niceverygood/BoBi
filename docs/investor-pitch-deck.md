## SLIDE 2 — Problem

**Headline**: 한국 보험 설계사 60만 명, 매일 똑같은 고통

**Body**:
1. **고객 분석에 시간이 너무 오래 걸린다**
   - 1명당 보장 분석 평균 **3시간** (수기 정리)
   - 청약서 작성 전 고지 사항 점검도 직접 (놓치면 보험금 거절 원인)

2. **클로징 무기가 부족하다**
   - "왜 이 보험이 필요한가" 설계사 말로만 설득
   - 고객 입장에서 "보장 갭이 얼마인지" 숫자로 보여주기 어려움

3. **영업 기회를 놓친다**
   - 갱신·면책·생일 등 핵심 컨택 타이밍을 수기로 관리
   - 1명 놓치면 평생 거래 끊김 (LTV 수백만원 손실)

4. **데이터·시스템 파편화**
   - HIRA 진료내역, 건강검진, 보험 가입 정보가 각각 따로
   - 통합 분석 도구가 시장에 부재

## SLIDE 3 — Solution: BoBi

**Headline**: 진료 데이터 + AI = 5분 컨설팅

**Body** (5대 핵심 기능):

| 기능 | 한 줄 설명 | 기술 기반 |
|---|---|---|
| **진료정보 분석** | 5년치 진료내역을 AI가 자동 정리 | HIRA 마이데이터 + LLM |
| **위험도 리포트** | 일반인 대비 개인 질환 위험 배율 | 진료+건검 데이터 통계 모델 |
| **가상 사고영수증** | 보장 갭을 숫자로 시각화 | 의료비 시뮬레이션 + 약관 매칭 |
| **CRM 자동 알림톡** | 갱신·면책·생일 자동 카톡 발송 | ALIGO 알림톡 API + 자체 cron |
| **미래의 나** | 연령대별 의료비 변화 시뮬레이션 | 통계청 데이터 + 시나리오 모델 |

**핵심 차별점**:
- 마이데이터 인증 사업자 공식 API → 합법적 진료 데이터 (경쟁사 대부분 미보유)
- 카카오 알림톡 발신자에 **설계사 본인 이름** → 설계사 브랜딩
- 보비 디자인시스템 §11.3 — 단정 표현 금지·정직 톤 (보험 산업 신뢰도 핵심)
