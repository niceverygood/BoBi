# 투자 제안서 — 한승수 입력 폼

`docs/investor-pitch-deck.md` 의 `[한승수 확인]` 표시된 빈칸을 한 곳에 모은
입력 폼. 1시간 내 채울 수 있도록 설계. 채운 뒤 같은 폴더의 deck 파일과
대조해서 v2 완성.

> 📌 **모르는 항목은 비워두세요.** Claude가 v2 작성 시 비어 있으면 "공개 시
> 보강 예정" 으로 우회하거나, 핵심 항목이면 "보수적 추정"으로 대체.
