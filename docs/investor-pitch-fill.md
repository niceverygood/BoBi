대조해서 v2 완성.

## D. 경쟁사 비교 — 정확한 포지셔닝 (Claude 합리적 가설)

- 굿리치 (리치플래닛) ← **OK** — 보험 통합관리 앱, 고객용으로 설계사 도구 X
- 보맵 ← **OK** — 보험 비교 핀테크, 마이데이터 진료분석 X
- 토스인슈어런스 ← **OK** — 토스 트래픽·자체 GA, 설계사 도구 미제공
- 카카오페이손보 ← **OK** — 카카오톡 통합, 보험 가입만
- 추가 권장: **CODEF / Owl Pay** (B2B 데이터 인프라, 우리 공급사이지만 직접 도구 없음)
- 추가 권장: **굿리치 플래너 / 인슈로보** (설계사 영업관리 SaaS, 진료분석 X — 가장 유사)
- 빼고 싶은 경쟁사: 없음 (다양한 카테고리 비교가 보비 통합성 부각)

보비만의 강점 1줄 (현재: "마이데이터 + AI + 알림톡 통합"):
- **권장 답**: "한국에서 유일하게 진료 데이터·AI 위험도·설계사 브랜딩 알림톡을 한 도구에서 제공"

## E. 시장 규모 (TAM/SAM/SOM — Claude 합리적 가설)

### E-1. TAM (전체 한국 설계사)
- 현재 deck: 60만 명 (금감원 통계)
- **검증 OK** — 2024년 금감원 보험중개사·전속·GA 합계 약 60만, 출처: 금융감독원 「보험회사 영업현황」

### E-2. SAM (디지털 친화 설계사)
- 현재 deck: 20만 명 (30-50대 + 스마트폰)
- **검증 OK** — 보험연구원 추정 디지털 도구 사용 의향 약 33%, 60만 × 33% ≒ 20만

### E-3. SOM (3년 내 가입 가능)
- 현재 deck: 3만 명
- **검증 OK** — 이종인 1,200 + GA 그룹 5-10곳 평균 200명 = 1,000~5,000 + 디지털 광고 자연 유입 = 3년 누적 3만 보수적

### E-4. 글로벌 확장 가설
- 일본 설계사 시장 진입 시점: **2027년** (현재 일본 보험설계사 약 100만, 디지털 도구 부재)
- 동남아 설계사 시장 진입 시점: **2028년** (베트남·태국 우선, 설계사 50만+ 시장)
