## 1. 환불 가능 케이스 (즉시 100% 환불)

| 케이스 | 조건 | 환불액 |
|---|---|---|
| **A. 결제 직후 미사용** | 결제일로부터 **7일 이내** + AI 분석 1회도 실행 안 함 | **100%** |
| **B. 자동 갱신 직후 미사용** | 자동 갱신일로부터 **3일 이내** + 갱신 후 분석 1회도 실행 안 함 | **100%** |
| **C. 회사 귀책 장애** | 시스템 장애·외부 API 장애로 서비스 이용 불가 (24시간 이상) | **잔여 기간 일할 환불** + 보상 검토 |
| **D. 부분 환불 요청 (사용 후)** | 결제일로부터 7일 이내 + 일부 분석 사용 | **이용 건수 차감 후 환불** (분석 1회당 베이직 663원/프로 1,330원) |

## 2. 환불 불가 케이스

- **결제일로부터 7일 경과** 후 (회사 귀책 장애 제외)
- **분석 결과 다운로드·고객 발송 완료** 건 (이미 가치 제공 완료)
- **약관 위반·부정 사용** (다중 계정 환불 악용, 결제 정보 위조 등)
- **무료체험 종료 후 정상 결제** 7일 경과 (체험 사용 자체로 청약철회 의사 철회로 간주)

## 3. 결제 수단별 환불 처리

### 3-1. 카카오페이 / 토스페이먼츠 / KG이니시스 (웹·신용카드)
- **보비가 직접 환불 처리** (1:1 문의 또는 마이페이지 구독 해지)
- 처리 기간: **영업일 3~5일 이내** 원 결제 수단으로 환급
- 카드 결제: 다음 카드 청구일 또는 해당월 청구서에 환불 반영

### 3-2. Apple App Store (iOS 인앱결제)
- **Apple이 환불 처리 권한** 보유 — 보비가 직접 환불 불가
- 사용자 직접 신청: https://reportaproblem.apple.com 또는 App Store > 구매 내역 > 환불 요청
- 보비가 보조: 환불 사유 작성 도움 + Apple 신청 가이드 제공
- Apple 처리 기간: 24~48시간 검토 + 환불 승인 시 영업일 5~10일

### 3-3. Google Play (Android 인앱결제)
- **48시간 이내 자동 환불 신청**: Google Play 앱 내 구매 내역 > 환불
- **48시간 경과**: 보비가 Play Console에서 환불 처리 가능 — 1:1 문의로 신청
- 처리 기간: 영업일 1~3일 이내 결제 수단으로 환급

## 4. 환불 신청 방법

### 옵션 A. 마이페이지 자체 해지 (자동 갱신 중지)
- 대시보드 > 설정 > 구독 관리 > "구독 해지"
- 즉시 자동 갱신 중지. 잔여 기간은 계속 이용 가능.
- ⚠️ 자동 갱신 중지 ≠ 환불. 즉시 환불 원할 시 옵션 B.

### 옵션 B. 1:1 문의로 환불 신청
- 대시보드 > 1:1 문의
- 또는 이메일: dev@bottlecorp.kr
- 필요 정보: 결제 일자 / 결제 수단 / 환불 사유
- 처리 회신: 영업일 1일 이내 (검토 후 승인 여부 안내)

### 옵션 C. 카카오톡 채팅 상담
- 평일 10:00~18:00
- 채팅 봇 → 환불 문의 선택 → 상담사 연결

## 5. 무료체험 환불 정책

- **무료체험 기간 (3일) 중 해지**: 청구 0원, 환불 사유 없음
- **무료체험 종료 후 자동 결제 직후 24시간 이내** + **분석 미사용**: 100% 환불
- **무료체험 종료 후 정상 사용 시작**: §1 일반 환불 정책 적용

## 6. 부분 환불 계산 공식

### 일할 계산 (회사 귀책·잔여 기간 환불)
```
환불액 = 결제 금액 × (잔여 일수 / 30일)

예시: 베이직 19,900원 결제 후 10일 사용 (잔여 20일)
     19,900 × (20/30) = 13,266원 환불
```

### 분석 건수 차감 (사용 후 7일 이내 환불)
```
환불액 = 결제 금액 - (사용 건수 × 건당 단가)

베이직 건당 단가: 19,900원 ÷ 30회 = 약 663원
프로 건당 단가:   39,900원 ÷ 무제한 → 일할 계산만 적용

예시: 베이직 결제 후 5건 분석 사용
     19,900 - (5 × 663) = 16,585원 환불
```

## 7. 자동 갱신·구독 해지 정책

### 자동 갱신
- 매월 결제일에 동일 금액 자동 청구
- 결제 24시간 전 카카오톡 알림 발송 (선택)
- 자동 갱신 해지 시 즉시 갱신 중지 (잔여 기간은 계속 이용)

### 청구 변경 정책
- 가격 인상 시: 시행일 30일 전 이메일·앱 내 공지 + 자동 갱신 동의 재확인
- 가격 인하 시: 즉시 적용
