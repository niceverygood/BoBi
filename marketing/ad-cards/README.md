# 보비 광고 카드뉴스 (인스타 1080×1080 캐러셀)

이종인 영업이사 5/9 카톡 — "비즈니스 계정으로 만들어서 메타광고로 올리면 될꺼야" — 후속 작업.
Meta Ads Manager에서 **앱 홍보 캠페인** 광고용 카드뉴스 8장.

## 구성

| # | 제목 | 목적 |
|---|---|---|
| s1 | 표지 — "설계사 1명이 한 달에 30시간을 이걸로 버린다" | 훅 (3초 안에 시선 잡기) |
| s2 | 문제 ① — 상담 1건 준비 평균 2시간 | 공감대 |
| s3 | 문제 ② — 엑셀 관리로 놓치는 영업 기회 월 200만원 | 공감대 + 손실 환산 |
| s4 | 해결 ① — 🎙️ 통화 녹음 → AI 자동 정리 | 가치 제안 |
| s5 | 해결 ② — 🏥 진료정보 5년치 + AI 고지의무 분석 | 가치 제안 |
| s6 | 해결 ③ — 📅 자동 알림 (갱신·면책·생일·1주년) | 가치 제안 |
| s7 | 사회적 증명 — 1,200명+ 사용 · ★4.8 · 후기 2건 | 신뢰 |
| s8 | CTA — 3일 무료 체험 · 카드 등록 없음 · App Store / Google Play | 전환 |

## 만들기

```bash
# 1. 미리보기 (브라우저)
open marketing/ad-cards/templates/slides.html

# 2. PNG 추출 (1080×1080 @2x = 2160×2160 실제 해상도)
node marketing/ad-cards/capture.js
```

`marketing/ad-cards/out/` 에 `s1.png ~ s8.png` 생성됨.

## 인스타 업로드

1. 인스타 비즈니스 계정 앱에서 → 게시물 만들기 → 캐러셀(여러 장 선택)
2. `s1.png` → `s2.png` → ... → `s8.png` 순서로 추가
3. 캡션은 `광고-카피.md` 의 3가지 버전 중 하나 사용
4. 게시 후 Meta Ads Manager에서 **게시물 홍보(Boost)** 또는 **앱 홍보 캠페인**의 광고 소재로 선택

## Meta Ads Manager에서 광고로 쓰기

1. Ads Manager → 캠페인 → 새 광고 만들기
2. **광고 형식**: 캐러셀 (Carousel)
3. **이미지**: s1~s8 각각 카드로 등록
4. **링크**: App Store / Google Play
5. **CTA 버튼**: "지금 설치" 또는 "더 알아보기"

## 디자인 수정

- 색상·폰트: `templates/slides.html` 상단 `:root` CSS 변수
- 후기 인용·이름: s6 (.feel), s7 (.review) 의 텍스트 직접 편집
- 1,200명 숫자 갱신: s7 의 `.big-num` 텍스트 변경
- 새 슬라이드 추가: `<div class="slide" id="s9">...</div>` 추가 후 `capture.js` 의 `SLIDES` 배열에 `'s9'` 추가

## 디자인 가이드라인 (수정 시 참고)
